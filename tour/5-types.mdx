		# VARIABLES AND TYPES

		In HookScript, you can declare your own variables and assign values to the them. When you declare a variable, you must declare its *type*, which tells HookScript what the variable is allowed to contain.

		There are *primitive* types and *container* types

		## Primitive types
		- `int` is used for integers (whole numbers)
		- `float` is used for floating point numbers (decimals)
		- `text` is used for character strings (letters and words)
		- `bool` is used for boolean values (true or false)
		
		## Container types
		- `map` a hashmap/object with key/value pairs
		- `[]int` an array (list) of `int`
		- `[]float` an array (list) of `float`
		- `[]text` an array (list) of `text`
		- `[]bool` an array (list) of `bool`
		- `[]map` an array (list) of `map`

		To declare a variable, you use the `let` keyword, give the variable a name, specify the variable's type, and assign a value

		```
		let x int = 10
		let y float = 10.1
		let a text = "hello world"
		let b = true
		```

		To change a variable's value, you use the assignment operator ( `=` ) and provide the correct type for the variable:

		```
		let x int = 10
		x = 20
		```

		Assigning an incorrect type to an existing variable will throw an error.

		We have updated the editor to use our new understanding of variables. Looking back at our **CONFIG** clause we see that it returns a **map** type. With this understanding, you can build
		more complex logic into your **CONFIG** clause, so long as it returns a **map** type.
